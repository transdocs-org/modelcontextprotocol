## 客户端详情

### 5ire

[5ire](https://github.com/nanbingxyz/5ire) 是一款开源的跨平台桌面 AI 助手，通过 MCP 服务器支持工具调用。

**主要特性：**

- 内置 MCP 服务器可快速启用/禁用。
- 用户可通过修改配置文件添加更多服务器。
- 开源且易于上手，适合初学者。
- 将持续改进对 MCP 的支持。

### AgentAI

[AgentAI](https://github.com/AdamStrojek/rust-agentai) 是一个 Rust 库，用于简化 AI Agent 的创建。该库内置与 MCP 服务器的无缝集成。

[MCP 服务器集成示例](https://github.com/AdamStrojek/rust-agentai/blob/master/examples/tools_mcp.rs)

**主要特性：**

- 多 LLM – 支持大多数 LLM API（OpenAI、Anthropic、Gemini、Ollama 及所有兼容 OpenAI API 的服务）。
- 内置 MCP 服务器支持。
- 使用类型安全、内存安全的 Rust 构建智能体工作流。

### AgenticFlow

[AgenticFlow](https://agenticflow.ai/) 是一个无代码 AI 平台，可帮助您构建 7×24 小时处理销售、营销和创意任务的智能体。通过 MCP 安全连接 2,500+ API 和 10,000+ 工具。

**主要特性：**

- 无代码 AI 智能体创建与工作流构建。
- 通过 MCP 访问 10,000+ 工具和 2,500+ API 的庞大库。
- 只需 3 步即可连接 MCP 服务器。
- 安全管理连接，随时撤销访问权限。

**了解更多：**

- [AgenticFlow MCP 集成](https://agenticflow.ai/mcp)

### AIQL TUUI

[AIQL TUUI] 是一款原生跨平台桌面 AI 聊天应用，支持 MCP。它支持多种 AI 提供商（如 Anthropic、Cloudflare、Deepseek、OpenAI、Qwen）、本地 AI 模型（通过 vLLM、Ray 等）以及聚合 API 平台（如 Deepinfra、Openrouter 等）。

**主要特性：**

- **动态 LLM API 与智能体切换**：可在不同 LLM API 和智能体之间即时切换。
- **全面能力支持**：内置支持工具、提示、资源和采样方法。
- **可配置智能体**：通过智能体设置灵活选择并自定义工具。
- **高级采样控制**：修改采样参数并利用多轮采样获得最佳结果。
- **跨平台兼容**：完全兼容 macOS、Windows 和 Linux。
- **免费且开源（FOSS）**：宽松许可证，允许修改和自定义应用打包。

**了解更多：**

- [TUUI 文档](https://www.tuui.com/)
- [AIQL GitHub 仓库](https://github.com/AI-QL)

### Amazon Q CLI

[Amazon Q CLI](https://github.com/aws/amazon-q-developer-cli) 是一款开源的终端智能编程助手。

**主要特性：**

- 完整支持 MCP 服务器。
- 使用喜欢的文本编辑器编辑提示。
- 通过 `@` 快速访问已保存的提示。
- 直接在终端中控制和管理 AWS 资源。
- 工具、配置文件、上下文管理、自动压缩等功能一应俱全！

**开始使用**
```bash
brew install amazon-q
```
### Amazon Q IDE

[Amazon Q IDE](https://aws.amazon.com/q/developer) 是一款开源、具备代理能力的 IDE 编码助手。

**主要功能：**

- 支持 VSCode、JetBrains、Visual Studio 和 Eclipse 等 IDE。
- 可直接在 IDE 内控制和组织 AWS 资源。
- 通过 IDE 用户界面为每个 MCP 工具管理权限。

### Apify MCP Tester

[Apify MCP Tester](https://github.com/apify/tester-mcp-client) 是一个开源客户端，通过 Server-Sent Events（SSE）连接任意 MCP 服务器。  
它是一个独立的 Apify Actor，专为通过 SSE 测试 MCP 服务器而设计，支持 Authorization 头。  
使用纯 JavaScript（传统风格）编写并托管在 Apify 上，无需任何设置即可运行。

**主要功能：**

- 通过 SSE 连接任意 MCP 服务器。
- 与 [Apify MCP Server](https://apify.com/apify/actors-mcp-server) 协作，可交互一个或多个 Apify [Actor](https://apify.com/store)。
- 根据上下文和用户查询动态使用工具（若服务器支持）。

### Amp

[Amp](https://ampcode.com) 是 Sourcegraph 构建的代理编码工具。  
它可在 VS Code（以及 Cursor、Windsurf、VSCodium 等兼容分支）中运行，也可作为命令行工具使用。  
它还支持多人协作——你可以共享线程并与团队协同工作。

**主要功能：**

- 对启用工具及权限进行细粒度控制  
- 支持在 VS Code 的 `mcp.json` 中定义的 MCP 服务器

### Augment Code

[Augment Code](https://augmentcode.com) 是一款面向 VS Code 和 JetBrains 的 AI 驱动编码平台，具备自主代理、聊天和补全功能。  
本地与远程代理均拥有完整的代码库感知能力，并原生支持 MCP，可通过外部来源和工具增强上下文。

**主要功能：**

- 本地与远程代理均完整支持 MCP。
- 通过 MCP 服务器添加额外上下文。
- 利用 MCP 工具自动化开发工作流。
- 适用于 VS Code 和 JetBrains IDE。

### BeeAI Framework

[BeeAI Framework](https://i-am-bee.github.io/beeai-framework) 是一个开源框架，用于大规模构建、部署和运行强大的代理工作流。  
该框架包含 **MCP Tool**，这是一项原生功能，可简化将 MCP 服务器集成到代理工作流中的过程。

**主要功能：**

- 将 MCP 工具无缝集成到代理工作流中。
- 从已连接的 MCP 客户端快速实例化框架原生工具。
- 计划未来支持代理式 MCP 能力。

**了解更多：**

- [在代理工作流中使用 MCP 工具的示例](https://i-am-bee.github.io/beeai-framework/#/typescript/tools?id=using-the-mcptool-class)

### BoltAI

[BoltAI](https://boltai.com) 是一款原生、一体化的 AI 聊天客户端，支持 MCP。  
BoltAI 支持多种 AI 提供商（OpenAI、Anthropic、Google AI…），包括本地 AI 模型（通过 Ollama、LM Studio 或 LMX）。
